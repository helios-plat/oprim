# Statistics Operations

## ops.bootstrap_ci
Non-parametric bootstrap confidence interval (percentile, BCa, basic methods).

## ops.percentile_ci
Compute percentile-based CI from sample array.

## ops.distribution_summary
Unified distribution descriptive statistics (mean, std, skew, kurtosis, percentiles).

## ops.skew_kurt_robust
Robust skewness + excess kurtosis with Fisher-Pearson correction.

## ops.kolmogorov_smirnov_test
KS test for distribution similarity (one-sample and two-sample).

## ops.mann_kendall_trend
Mann-Kendall monotonic trend test with Hamed-Rao autocorrelation correction.

## ops.bayes_beta_update
Beta(α, β) posterior update for Thompson Sampling.

## ops.brier_score_decomposed
Brier Score Murphy 1973 three-component decomposition.

## ops.pearson_spearman_corr
Pearson + Spearman correlation with p-values.

## ops.kde_density
Kernel density estimation with Silverman/Scott bandwidth.
