# Distance Operations

## ops.wasserstein_distance
Wasserstein distance (1D exact via scipy, sliced multi-D approximation).

## ops.dtw_distance
Dynamic Time Warping with Sakoe-Chiba band constraint.

## ops.cosine_similarity_batch
Batch cosine similarity with optional top-k selection.

## ops.euclidean_distance_matrix
Pairwise Euclidean distance matrix with optional feature weights.

## ops.symmetric_kl_divergence
Jensen-Shannon divergence or symmetric KL divergence.
